CSS Grid and Flexbox are layout models that share similarities and can be used together. The key difference is that CSS Grid can be used to create two-dimensional layouts, while Flexbox can only be used to create one-dimensional layouts. That means you can place components along the X- and Y-axis in CSS Grid and only one axis in Flexbox. So we can say that, grid is useful for making a section's overall layout while flex box is conventionally used to position elements of a particular part inside the section.

The main difference between Bootstrap and Tailwind is that Bootstrap pre-built components that provide a responsive UI (user interface) kit. On the other hand, Tailwind comes with predesigned widgets that facilitate web developers to design a website from scratch . Both Bootstrap and Tailwind are two of the best front-end CSS framework used worldwide. Bootstrap is the most popular framework till the current date and is more suitable for begginers and tailwind is is used to execute detailed style modifications.

The components that can be depicted on the web page consist of one or more than one rectangular box. A CSS box model is a compartment that includes numerous assets, such as edge, border, padding and material. It is used to develop the design and structure of a web page. It can be used as a set of tools to personalize the layout of different components. According to the CSS box model, the web browser supplies each element as a square prism. For further info you can visit MDN docs.

The HTML semantics refers to the tags that provide meaning to an HTML page rather than just presentation. It makes HTML more comprehensible by better defining the different sections and layout of web pages. For example:
header: defines a header for a web page.
nav:defines a container for navigation links.
